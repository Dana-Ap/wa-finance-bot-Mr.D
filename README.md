# 🤖 WhatsApp Personal Finance Bot (Mr.D Edition)

Asisten keuangan pribadi otomatis berbasis WhatsApp yang dikembangkan menggunakan **Google Apps Script** dan **Fonnte API**. Bot ini dirancang untuk memudahkan pencatatan transaksi harian secara cepat dengan fitur konfirmasi interaktif.

## 🚀 Fitur Utama
- **Jalur Turbo:** Pencatatan super cepat untuk pengeluaran harian.
- **Konfirmasi Interaktif:** Menanyakan kategori secara otomatis untuk transaksi besar atau tidak dikenal.
- **Laporan Otomatis:** Cek saldo harian, bulanan, dan laporan gabungan (joint account) langsung via chat.
- **Anti-Spam:** Sistem cache untuk mencegah duplikasi transaksi.
- **Auto-Formatting:** Sinkronisasi format data otomatis ke Google Sheets.

## 🛠️ Stack Teknologi
- **Bahasa:** JavaScript (Google Apps Script)
- **Database:** Google Sheets API
- **Messaging:** Fonnte API (WhatsApp Gateway)

## 📋 Cara Penggunaan
Cukup kirim pesan ke bot dengan format:
- `masuk 5jt Gaji`
- `keluar 15rb Jajan Bakso`
- `laporan` (untuk melihat ringkasan)
